    1、介绍：

    xxl-job是一个分布式任务调度平台，在分布式环境下可以自行控制任务调度时间、监控、可在不同环境下随意设置配置；

    整个任务调度由两部分组成：

    1、admin程序：用于配置各个任务的调度时间、监控等。此程序一定要注意安全，不允许其他人随意访问；

    2、应用程序：具体的任务执行程序；

    请注意：admin程序版本号，必须与应用程序版本号一致，否则可能出现各种各样的问题！文章使用的版本为：2.3.1

    2、部署admin程序：

    我们使用docker的形式部署，好处是方便！而且它身为一个调度服务，也并不需要特别大的jvm内存；

    docker地址：https://hub.docker.com/r/xuxueli/xxl-job-admin

    docker run -e JAVA_OPTS="-Xmx512m" -e PARAMS="--spring.datasource.url=URL --spring.datasource.username=USERNAME --spring.datasource.password=PASSWORD --spring.mail.host=HOST --spring.mail.port=PORT --spring.mail.username=USERNAME --spring.mail.from=FROM --spring.mail.password=PASSWORD " -p 9091:8080 -v /data/logs/xxl-dev:/data/applogs --name xxl-job-admin-dev  -d xuxueli/xxl-job-admin:2.3.1
    • -e JAVA_OPTS：JVM基础配置，我们这里只配置了堆内存的最大限制；
    • -e PARAMS：java程序的配置信息，也就是application.yml中写的；
      • –spring.datasource.url：数据库链接地址；
      • spring.datasource.username：用户名；
      • spring.datasource.password：密码；
      • spring.mail.host：报警邮件；
      • spring.mail.port：报警邮件端口；
      • spring.mail.username：邮件用户名；
      • spring.mail.from：邮件发送者；
      • spring.mail.password：邮件密码
    • –name：当前admin程序的docker镜像名

    3、部署应用程序

    1、增加maven地址

            <dependency>
            <groupId>com.xuxueli</groupId>
            <artifactId>xxl-job-core</artifactId>
            <version>2.3.1</version>
        </dependency>

    2、增加application.yml配置

    xxl:
  job:
    admin:
      # admin程序启动地址，用于主动注册
      addresses: http://192.168.20.204:9091/xxl-job-admin
    # 与admin程序配套的token，不一致会被拒绝
    accessToken: default_token
    executor:
      # 应用名
      appname: caocao-service
      address:
      ip:
      # 0为默认端口9999
      port: 0
      logpath: /data/applogs/xxl-job/jobhandler
      logretentiondays: 30

    3、增加配置文件

    @Configuration
public class XxlJobConfig {
    private Logger logger = LoggerFactory.getLogger(XxlJobConfig.class);

    @Value("${xxl.job.admin.addresses}")
    private String adminAddresses;
    @Value("${xxl.job.accessToken}")
    private String accessToken;
    @Value("${xxl.job.executor.appname}")
    private String appname;
    @Value("${xxl.job.executor.address}")
    private String address;
    @Value("${xxl.job.executor.ip}")
    private String ip;
    @Value("${xxl.job.executor.port}")
    private int port;
    @Value("${xxl.job.executor.logpath}")
    private String logPath;
    @Value("${xxl.job.executor.logretentiondays}")
    private int logRetentionDays;


    @Bean
    public XxlJobSpringExecutor xxlJobExecutor() {
        logger.info(">>>>>>>>>>> xxl-job config init.");
        XxlJobSpringExecutor xxlJobSpringExecutor = new XxlJobSpringExecutor();
        xxlJobSpringExecutor.setAdminAddresses(adminAddresses);
        xxlJobSpringExecutor.setAppname(appname);
        xxlJobSpringExecutor.setAddress(address);
        xxlJobSpringExecutor.setIp(ip);
        xxlJobSpringExecutor.setPort(port);
        xxlJobSpringExecutor.setAccessToken(accessToken);
        xxlJobSpringExecutor.setLogPath(logPath);
        xxlJobSpringExecutor.setLogRetentionDays(logRetentionDays);
        return xxlJobSpringExecutor;
    }
}
